# agency-os

> The complete Claude Code toolkit for web & advertising design agencies.

**Work in progress — v1.0.0 coming soon.**

```
/plugin marketplace add gekkos-tech/agency-os
```

## Bundles

| Plugin | Purpose |
|---|---|
| `agency-core` | Project workflow router (BRIEF → CONCEPT → DESIGN → BUILD → QA → LAUNCH), handoffs, proposals |
| `design-foundation` | UI/UX quality layer, taste, style systems, design review |
| `motion-3d` | GSAP suite, Three.js/WebGL, React Three Fiber, Lottie, Rive, Remotion |
| `quality-qa` | Performance, Core Web Vitals, accessibility, testing, pre-launch checks |
| `seo-marketing` | SEO audits, technical SEO, content, schema, local SEO, AI visibility |
| `ad-creative` | Campaign ideation, ad copy, social formats, landing page CRO, brand kits |
