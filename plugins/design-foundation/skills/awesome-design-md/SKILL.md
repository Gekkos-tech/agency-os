# Awesome DESIGN.md — brand design-system reference library

A curated collection of `DESIGN.md` files, each a deep analysis of one real brand's
visual language: color tokens, typography scale, spacing, surfaces, component patterns,
and motion. Source: VoltAgent/awesome-design-md.

## How to use

1. Pick the brand closest to the target aesthetic from the index below.
2. Read its `DESIGN.md` (and `README.md` if present) from
   `reference design-md/<brand>/DESIGN.md` in this skill folder.
3. Treat that file as the ground-truth design spec — apply its tokens (colors, type,
   spacing, radii, shadows, motion) to the user's project. Adapt, don't copy verbatim:
   respect the user's existing brand and content.

For a new project, a brand's `DESIGN.md` can be copied into the repo root as a
starting `DESIGN.md` and refined (this pairs well with the `impeccable` and taste skills).
