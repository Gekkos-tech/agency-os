# YouTube — Placement Specs

Specs verified July 2026 — always spot-check platform docs before final export.
Sources: Google Ads / YouTube Help + adnabu.com, quickframe.com, snappa.com
spec roundups (retrieved 2026-07). Video ads are served from an uploaded YouTube
video (public or unlisted), so YouTube upload specs apply.

## Shorts Ads

| Attribute | Spec |
|---|---|
| Ratio / px | 9:16 → 1080x1920 recommended; 1:1 and 16:9 accepted (letterboxed) |
| Duration | up to 3 min possible; ≤ 60 s recommended |
| Note | Served between organic Shorts; design like TikTok/Reels — hook in 2 s |

## Skippable In-Stream Ads

| Attribute | Spec |
|---|---|
| Ratio / px | 16:9 → 1920x1080; 9:16 → 1080x1920; 1:1 → 1080x1080 |
| Duration | any length accepted; skip button at 5 s; 15–20 s (awareness) to ~3 min (consideration) recommended |
| File size | ≤ 256 GB (YouTube upload limit) |
| Formats | MP4 recommended (MPG, MOV, AVI, WMV, WebM, 3GPP, ProRes, HEVC accepted) |
| Companion banner | 300x60 px, JPG/GIF/PNG, ≤ 150 KB (desktop only) |
| Critical rule | Brand + message inside first 5 s — assume the skip |

## Non-Skippable In-Stream Ads

| Attribute | Spec |
|---|---|
| Ratio / px | 16:9 / 9:16 / 1:1, same px as skippable |
| Duration | 7–15 s standard (regional); up to 30 s on eligible CTV placements |
| File size / formats | Same as skippable |

## Bumper Ads

| Attribute | Spec |
|---|---|
| Duration | ≤ 6 s — hard limit |
| Ratio / px | 16:9 / 9:16 / 1:1, same px as in-stream |
| File size | ≤ 1 GB |
| Use | One message, one visual, end-card logo |

## In-Feed Video Ads (thumbnail + text in feed/search)

| Attribute | Spec |
|---|---|
| Video | Any public YouTube video; 16:9 / 9:16 / 1:1 recommended |
| Headline | ≤ 100 chars |
| Description | 2 lines, ≤ 35 chars each |

## Thumbnails

| Attribute | Spec |
|---|---|
| Size | 1280x720 (16:9); min width 640 px |
| Formats | JPG, PNG, GIF |
| Max file size | 2 MB |
| Text rule | ≤ 3–4 words, legible at 120 px wide (search result size) |

## Channel Banner

| Attribute | Spec |
|---|---|
| Upload size | 2560x1440 |
| Safe area (all devices) | center 1546x423 — ALL text/logos inside this box |
| Device crops | mobile shows only 1546x423; desktop ~2560x423; TV full 2560x1440 |
| Formats / size | JPG, PNG, GIF, BMP; ≤ 6 MB |

## Masthead (reservation)

| Attribute | Spec |
|---|---|
| Resolution | 1920x1080 or higher, 16:9 |
| Autoplay | ≤ 30 s muted autoplay on desktop |
| Note | Reservation only via Google sales; use the Masthead Preview Tool |

## Safe zones — vertical (9:16, 1080x1920) in-stream/Shorts

| Edge | Keep clear | UI element |
|---|---|---|
| Top | ~288 px | Status/channel info |
| Bottom | ~672 px | CTA, ad badge, engagement panel |
| Right | ~192 px | Like/share rail (Shorts) |
| Left | ~48 px | Margin |

Horizontal in-stream: keep bottom ~387 px and right ~162 px clear of critical
content (overlay CTA + ad UI), top ~38 px.

## 2025/2026 changes noted

- Shorts ads mainstreamed as a distinct placement with up-to-3-min ceiling.
- Vertical (9:16) creative now first-class across in-stream formats.
- Pause Ads (CTV) added as a format; specs via Google sales.
